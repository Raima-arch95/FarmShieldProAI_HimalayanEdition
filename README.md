Farm Shield AI

Overview
--------
PyTorch MobileNetV3-based plant disease classifier for Farm Shield Pro.

Supported Crops
---------------
• Apple
• Potato
• Tomato
• Maize
• Wheat
• Capsicum

Dataset
-------
40,977 images
35 classes

Model
-----
MobileNetV3 Small (Transfer Learning)

Validation Accuracy
-------------------
91.3%

Deployment
----------
TensorFlow Lite (.tflite)
