# Farm Shield Dataset Mapping

This document defines the mapping between the original PlantVillage dataset
folder names and the standardized Farm Shield dataset structure.

The standardized names are used throughout the AI training pipeline,
TensorFlow Lite model, and Flutter application.

---

# Apple

| PlantVillage Folder | Standard Crop | Standard Disease |
|---------------------|---------------|------------------|
| Apple___healthy | Apple | Healthy |
| Apple___Apple_scab | Apple | AppleScab |
| Apple___Black_rot | Apple | BlackRot |
| Apple___Cedar_apple_rust | Apple | CedarAppleRust |
